## ğŸ‘ï¸ For Those With Visual Impairments

### Screen Reader Friendly
- All images have `alt` text
- Documentation is plain Markdown (screen reader compatible)
- No essential information in images only

### High Contrast Mode
- Our SVGs use high-contrast colors
- Text is always readable
- No information conveyed by color alone

### Text-Based Play
You can play entirely via terminal:
```bash

git clone https://github.com/fabriziosalmi/enjoy.git
cd enjoy

# Create your contribution
echo "YOURWORD" > yourword.txt

# Submit
git add . && git commit -m "Add YOURWORD" && git push
```
